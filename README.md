# GPL Raider

Turning GPL-neglected ideas into practical free software.

I build small, useful tools for Android and Windows with a bias toward source-available workflows, reproducible packaging, and no advertising or analytics.

## Projects

- [Ventoid](https://github.com/GPLaider/Ventoid) - Android OTG Ventoy-style USB writer. GPL-3.0-only, available on [F-Droid](https://f-droid.org/packages/com.ventoid.app/).
- [zipzip](https://github.com/GPLaider/zipzip) - Windows-first Korean-friendly archive manager built with WinUI 3.

## Principles

- GPL means source should be usable, buildable, and reviewable.
- Practical free software beats polished lock-in.
- No ads, no analytics, no unnecessary network dependency.
